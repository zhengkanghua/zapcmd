### Cross-Phase Integration Issues

- **Phase 10 docs → `scripts/verify-local-gate.mjs`:** 文档声称 macOS 默认执行 desktop smoke，但当前脚本仍把 macOS 放在实验开关后。
- **Phase 10 docs → GitHub workflows:** 文档 / verification 声称 CI / Release 存在 macOS desktop smoke 阻断，但当前 workflow 仍只有 Windows desktop smoke job。

### Broken Flows

- **macOS local verification flow:** breaks at `npm run verify:local`，因为默认路径仍跳过 desktop smoke。
- **macOS remote gate flow:** breaks in GitHub Actions，因为当前并不存在真实的 macOS desktop smoke 阻断 job。

- macOS desktop smoke 的稳定性与口径统一仍需通过 Phase 12 收敛。
