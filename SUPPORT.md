# Support

## Get Help

1. Read `README.md` / `README.zh-CN.md`.
2. Check `docs/README.md`.
3. Search existing issues first.

## Bug Report Template

Include:

1. OS + architecture.
2. ZapCmd version/tag.
3. Reproduction steps.
4. Expected vs actual behavior.
5. Logs/screenshots (if available).

## Feature Request Template

Include:

1. Problem statement.
2. Proposed behavior.
3. Tradeoffs or alternatives.

## Support Scope

Maintainers prioritize:

1. Reproducible bugs in current release.
2. Security/stability issues.
3. Critical regressions in core workflow.

Support is best-effort; response time may vary.
